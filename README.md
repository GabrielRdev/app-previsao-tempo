# App de Previsão do Tempo


### 🛠 Projeto
O App Previsão do Tempo é um projeto que mostra ao usuário a previsão do tempo de qualquer lugar do mundo através de uma API gratuíto da Open Weather.

### 🛠 Tecnologias

- React JS
- Axios
- CSS
- Git
